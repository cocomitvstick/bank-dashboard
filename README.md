# ПСБ Analytics — Аналитика банковского сектора РФ

Интерактивный аналитический дашборд для сравнения **АО «ПРИМСОЦБАНК»** с другими банками России.

---

## Быстрый старт

### Требования
- Python 3.10+
- Интернет (для загрузки данных с ЦБ РФ и MOEX)

### Установка и запуск

```bash
# 1. Клонируйте или распакуйте проект
cd bank-dashboard

# 2. Запустите одним файлом (Windows)
run.bat

# 3. Откройте в браузере
# http://localhost:8000/app — дашборд
# http://localhost:8000/docs — API документация
```

**Linux / macOS:**
```bash
pip install -r backend/requirements.txt
cd backend
python main.py
```

## Описание проекта

| Параметр | Значение |

| Регион | Приморский край |

### Что показывает дашборд

- **KPI-карточки** — активы, капитал, прибыль, норматив Н1, место в рейтинге РФ
- **Линейный/столбчатый график** — динамика показателя в сравнении с группами банков
- **Радар-диаграмма** — профиль по 6 показателям vs ТОП-10 среднее
- **Рейтинговая таблица** — место среди 150+ банков с дельтой к предыдущему периоду
- **Сравнение групп** — горизонтальные бары по группам (ТОП-10/50/100/все банки)

## Источники данных

| Источник | URL | Обновление |
|----------|-----|-----------|
| ЦБ РФ — Формы 101, 102 | cbr.ru/banking_sector/otchetnost/ | Ежемесячно (~15-е число) |
| ЦБ РФ — OpenData CSV | cbr.ru/opendata/ | Ежемесячно |
| MOEX ISS API | iss.moex.com/iss | Real-time / ежедневно |

### Задержка данных ЦБ

Данные за **январь** публикуются приблизительно **15 февраля**. Приложение учитывает эту задержку при определении «последнего доступного» периода.

## API

После запуска сервера доступны эндпоинты:

```
GET  /api/status                    — статус данных
GET  /api/banks                     — список банков
GET  /api/banks/top?n=50            — ТОП-N по активам
GET  /api/banks/{reg}/metrics       — показатели банка
GET  /api/banks/{reg}/timeseries    — временной ряд
GET  /api/compare                   — сравнение банков/групп
GET  /api/groups/averages           — средние по группам
GET  /api/radar/{reg}               — данные для радар-диаграммы
GET  /api/rankings                  — рейтинговая таблица
GET  /api/moex/keyrate              — ключевая ставка ЦБ
GET  /api/metrics/list              — справочник показателей
POST /api/refresh                   — принудительное обновление
```

Полная документация: **http://localhost:8000/docs**

## Перенос на другой ПК

### Что нужно установить

1. **Python 3.10+** — скачать с [python.org](https://www.python.org/downloads/)
   При установке поставить галочку **«Add Python to PATH»**

2. **Зависимости Python** — выполнить в папке проекта:
   ```bat
   pip install -r backend\requirements.txt
   ```

### Как перенести проект

```
Способ А — архив:
  1. Заархивировать папку bank-dashboard/ (ZIP или RAR)
  2. Скопировать архив на новый ПК (флешка, сеть, почта)
  3. Распаковать в любую папку, например C:\Apps\bank-dashboard\
  4. Открыть командную строку в этой папке
  5. pip install -r backend\requirements.txt
  6. Запустить run.bat

Способ Б — сетевая папка / NAS:
  Приложение работает прямо из сетевой папки, если Python установлен.
  Любой ПК в сети может запустить свою копию.
```

### Запуск как сервер (доступ с любого ПК)

Чтобы дашборд был доступен с нескольких компьютеров одновременно — запустить
на одном ПК-сервере, остальные подключаются через браузер по его IP:

```bat
REM На сервере (узнать IP: ipconfig)
cd backend
python main.py --host 0.0.0.0 --port 8000
```

```
REM На клиентских ПК — открыть браузер и зайти:
http://192.168.1.XXX:8000/app
```

> Или добавить в `run.bat` строку `python main.py --host 0.0.0.0 --port 8000`

### Требования к сети (для информационной безопасности)

Дашборд обращается к следующим внешним ресурсам. Все соединения — только
**исходящие (outbound) HTTPS (TCP 443)**. Входящего трафика из интернета
приложение не требует.

| Ресурс | Домен | Для чего |
|--------|-------|---------|
| ЦБ РФ — банковская отчётность | `www.cbr.ru` | Форма 101, список банков, ставки |
| ЦБ РФ — SOAP | `www.cbr.ru` | Данные по банкам (формы 101, 135) |
| ЦБ РФ — курсы валют | `www.cbr.ru` | USD/EUR/CNY (официальный курс) |
| ЦБ РФ — учётные цены металлов | `www.cbr.ru` | Золото, серебро, платина, палладий |
| Московская биржа ISS API | `iss.moex.com` | Индексы, акции, валюты, ключевая ставка |

**Если интернет заблокирован:** приложение автоматически переходит в
**демо-режим** — данные рассчитываются по модели, индикатор в шапке
станет оранжевым «Демо-данные».

**Минимум для работы в защищённой сети:**
- Разрешить исходящий HTTPS на `www.cbr.ru` и `iss.moex.com`
- Или развернуть прокси-сервер, который кэширует эти API

### Оффлайн-режим

Приложение кэширует все загруженные данные в `backend/data/`.
После первого запуска с интернетом кэш сохраняется — можно работать
без интернета, пока кэш актуален (обычно 1 месяц).

## Режимы работы

### Нормальный режим
Приложение загружает реальные данные с ЦБ РФ и MOEX. Данные кэшируются локально в `backend/data/`.

### Демо-режим
Если ЦБ недоступен (нет интернета, сайт лежит), приложение автоматически переходит в демо-режим. Дашборд показывает **расчётные данные**, основанные на публично известных приблизительных значениях. В UI появляется предупреждение.
